What Does Responsive Web Design Solve?

Before reading the first chapter of Responsive Web Design it would have been difficult for me to think of any problems that responsive design solves, likely because responsive web design is so heavily integrated in to my daily use of the web. In a brief search of responsive web design you can find a photo that shows water conforming to varying sizes of containers, proclaiming “content is like water!” This is the perfect way to understand how RSD solves the former problem of conforming content to varying screen sizes and varying platforms. In my house alone, we have multiple Apple products, a PC and a Playstation that connect to the web. Being able to see content that is standardized and functional on multiple gadgets greatly improves my user experiences, as I’m sure it does for others.
